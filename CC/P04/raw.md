## Theory

### Salesforce Platform Overview

Salesforce is the primary enterprise offering within the Salesforce platform. It provides:

- Case management and task management.
- Automated routing and escalation of important events.
- Customer portal for tracking cases.
- Social networking integration.
- Analytical tools and services such as email alerts, Google search, and access to entitlements and contracts.

### Lightning Platform (Force.com)

Lightning Platform (Force.com) is a PaaS that allows developers to create add-on applications that integrate into Salesforce. These applications are hosted on Salesforce infrastructure and are built using declarative tools, Lightning, Apex, and Visualforce.

### Community Cloud

Community Cloud enables external collaboration, customer service, channel sales, and custom portals. It is tightly integrated with Sales Cloud, Service Cloud, and App Cloud.

### Salesforce Sales Cloud

Sales Cloud is a CRM platform designed for sales, marketing, and customer support in both B2B and B2C contexts.

## Create Custom Apps for Salesforce Classic

Create custom apps to give Salesforce Classic users access to everything they need in one place. If you're new, use Lightning Platform quick start to create a basic working app in one step.

If you already created the objects, tabs, and fields, follow these steps to create an app label, logo, and assign it to profiles:

1. From Setup, enter **Apps** in the Quick Find box, then select **Apps**.
2. Click **New**.
3. If Salesforce console is available, select custom app or Salesforce console.
4. Give the app a name and description (max 40 characters).
5. Optionally add a custom logo.
6. Select items to include in the app.
7. Optionally set the default landing tab.
8. Choose which profiles the app will be visible to.
9. Check **Default** to make it the profile’s default app.
10. Click **Save**.
