## Assignment No. 5 (Mini Project)

### Aim / Title

Set up your own cloud for Software as a Service (SaaS) over the existing LAN in your laboratory. Write code for a cloud controller using open-source technologies to implement an HDFS-style workflow. Basic operations include dividing a file into segments/blocks and uploading/downloading the file in encrypted form.

### Objectives

- Set up your own cloud for SaaS over the existing LAN.
- Implement basic operations like file segmentation and upload/download.

## Theory

This practical simulates basic HDFS behavior by splitting a file into blocks, storing them separately, and reconstructing the file on download. A simple reversible transformation is used to represent encryption and decryption.

---

## Output

The program prints upload and download status, lists the created blocks, and creates a reconstructed output file.

## Conclusion

Thus, we implemented a mini cloud storage system that demonstrates file segmentation, encrypted block storage, and file reconstruction in a SaaS-style workflow.
