# LP-II Practicals Repository

**Practical exam scope:** From AI, only P01 to P05 are included. From CC, only P02 to P04 are included. CC P01 and CC P05 are still available in this repo for reference.

## Overview

This repository contains lab practicals for the LP-II course, organized into two main tracks:

- **AI**: Artificial Intelligence practicals (P01 to P06)
- **CC**: Cloud Computing practicals (P01 to P05)

Each practical folder typically contains:

- `raw.md`: Formatted write-up or theory
- `instructions.md` or `INSTRUCTIONS.MD`: Step-by-step execution guide
- `viva.md`: Viva questions and answers
- Code files (where applicable)
